About This Audit Tool

What is this? DRUSCAN is an automated Drupal audit tool that generates comprehensive technical analysis without requiring direct database or codebase access. Perfect for evaluating sites when transitioning between agencies, getting multiple vendor quotes, or assessing technical quality and security posture.

Primary Use Case: Agency Transitions & Vendor Evaluation

Project Handover Without Security Risks

When transitioning a Drupal site to a new agency or getting quotes from multiple vendors, sharing full database dumps and complete codebase access creates security and confidentiality concerns. DRUSCAN solves this by:

What You Can Share Safely
  • Structured JSON data about system configuration
  • Module list and architecture overview
  • Performance and security metrics
  • Code quality statistics (without actual code)
  • Content structure and entity relationships
What Stays Protected
  • Actual database content and user data
  • Proprietary custom code and business logic
  • API keys and integration credentials
  • Private files and media assets
  • Server access and deployment details
Perfect For: Request quotes from 3-5 agencies simultaneously without exposing sensitive data. Each vendor gets complete technical insight needed for accurate estimation, while your intellectual property and data remain secure.

How It Works

1. Data Collection

Automated scripts collect data using Drush commands and database queries.

2. Analysis

AI-powered analysis processes the data to identify patterns and issues.

3. Report Generation

Comprehensive report with findings, metrics, and visualizations is generated.

4. Recommendations

Actionable recommendations prioritized by impact and effort required.

Navigation Tips
  • Use the left sidebar to navigate between sections
  • Each section provides detailed analysis and data for that specific area
  • The Executive Summary at the end provides high-level overview of findings
  • Check Key Recommendations for prioritized action items
  • Review the Conclusion for final assessment and next steps
Version Compatibility

This audit tool is designed for Drupal 8, 9, 10, and 11. It uses modern Drush commands and database schema from Drupal 8+. Drupal 7 sites are not supported by this automated audit tool.
